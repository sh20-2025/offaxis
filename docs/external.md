## Running our Project

1. Clone from Repository to your system
2. Go to Project Root Directory
3. Run
```powershell
./scripts/setup.ps1
```
4. Run
```powershell
./scripts/activate-venv.ps1
```
5. Run
```powershell
./scripts/check-django.ps1
```
6. Run
```powershell
./scripts/run-docker.ps1
```
7. Go to https://Localhost:8000
